

описание функций:

Структуры:

element - структура, представляющая элемент дека. Она содержит следующие поля:

value - значение элемента
next - указатель на следующий элемент в деке
prev - указатель на предыдущий элемент в деке
deck - структура, представляющая двусвязный дек. Она содержит следующие поля:

first - указатель на первый элемент дека
last - указатель на последний элемент дека
size - количество элементов в деке

Функции:

initDeck(deck *D) - инициализирует дек, устанавливая начальные значения полей.

size(deck *D) - возвращает количество элементов в деке.

empty(deck *D) - проверяет, пуст ли дек. Возвращает true если дек пуст, иначе false.

pushFront(deck *D, int value) - добавляет новый элемент со значением value в начало дека.

pushBack(deck *D, int value) - добавляет новый элемент со значением value в конец дека.

popFront(deck *D) - удаляет и возвращает значение первого элемента дека.

popBack(deck *D) - удаляет и возвращает значение последнего элемента дека.

topFront(deck *D) - возвращает значение первого элемента дека без удаления.

topBack(deck *D) - возвращает значение последнего элемента дека без удаления.

deleteDeck(deck *D) - освобождает память, занимаемую деком и его элементами.

cat(deck *D1, deck *D2) - объединяет два дека D1 и D2, возвращая новый дек, содержащий все элементы обоих деков.

append(deck *D1, deck *D2) - добавляет все элементы из дека D2 в конец дека D1.

printDeck(deck *D) - выводит содержимое дека на экран.

Функция printChoose() просто выводит меню с выбором действий на экран.
